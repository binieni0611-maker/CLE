## 🛒 가비아에서 도메인 구매하기 (단계별)

### STEP 1: 회원가입 (5분)
1. https://www.gabia.com 접속
2. 우측 상단 **회원가입** 클릭
3. 이메일, 비밀번호 입력
4. 본인 인증 (휴대폰)
5. 가입 완료

### STEP 2: 도메인 검색 (2분)
1. 메인 페이지 검색창 찾기
2. 원하는 도메인 입력 (예: `wellness-coaching`)
3. **검색** 버튼 클릭

### STEP 3: 도메인 선택 (3분)
가비아가 다음과 같이 보여줍니다:

```
wellness-coaching.com       등록가능    15,000원/년  [담기]
wellness-coaching.co.kr     등록가능    18,000원/년  [담기]
wellness-coaching.kr        등록가능    20,000원/년  [담기]
wellness-coaching.net       등록가능    16,000원/년  [담기]
```

- **초록색 "등록가능"**: 구매 가능 ✅
- **빨간색 "등록불가"**: 이미 다른 사람이 소유 ❌

**추천 순서:**
1. `.com` (전 세계적으로 가장 인기)
2. `.co.kr` (한국 기업 신뢰도 높음)
3. `.kr` (한국 도메인)

원하는 도메인 옆 **장바구니 담기** 클릭

### STEP 4: 옵션 선택 (2분)
- **등록 기간**: 1년 (추천) 또는 2-10년
- **자동 갱신**: 체크 (추천) - 만료 걱정 없음
- **도메인 보호**: 선택 (개인정보 보호)

**결제하기** 클릭

### STEP 5: 결제 (3분)
- **신용카드** 또는 **무통장입금**
- 연간 비용 결제 (1년: 15,000원~)

✅ **구매 완료!**

### 자주 묻는 질문

**Q1: 도메인 등록 후 바로 사용 가능한가요?**
- A: 구매 즉시 소유권 확보, Vercel 연결 후 1-2시간 후 사용 가능

**Q2: 도메인 이름을 나중에 바꿀 수 있나요?**
- A: 도메인 이름 자체는 변경 불가, 새 도메인 구매 후 이전 가능

**Q3: .com과 .co.kr 차이는?**
- `.com`: 전 세계 대상, 가장 인기
- `.co.kr`: 한국 기업, 신뢰도 높음, 사업자등록증 필요

**Q4: 도메인 소유권은 누구에게?**
- A: 구매한 본인, 가비아는 관리만 대행

**Q5: 갱신 안 하면?**
- A: 만료 후 약 30일 유예 기간, 이후 타인이 구매 가능
