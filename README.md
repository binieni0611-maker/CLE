# 웰니스코칭연구소 홈페이지

**THE 행복한 삶을 위한.. 건강한 몸과 마음, 행복한 삶을 위한 교육**

## 🌟 프로젝트 소개

웰니스코칭연구소의 공식 홈페이지입니다. 교육학 박사, 코칭심리사, 간호사 출신 전문가가 이끄는 신뢰할 수 있는 교육 기관으로, 이론과 실무를 겸비한 전문성을 바탕으로 다양한 교육 프로그램을 제공합니다.

## 🎯 주요 특징

- **전문성**: 교육학 박사(평생교육·인적자원개발 전공) + 코칭심리사 + 간호사 출신
- **통합적 접근**: 학문적 깊이와 실무 전문성의 완벽한 조합
- **신뢰감**: 현장 경험 기반의 검증된 교육 프로그램
- **맞춤 교육**: 생애주기별, 대상별 맞춤형 교육 설계

## 📚 교육 프로그램

### 1. 코칭 교육
- 코칭 교수법 (교육학 박사 전문성 활용)
- 셀프 코칭 (코칭심리 전공 기반)
- 부모교육 (발달심리와 코칭 접목)
- 의사소통 교육 (코칭 기반 소통법)

### 2. 교회 평생교육
- 생애발달주기별 교육 (평생교육 전공 기반)
- 시니어교육 (노년기 발달과 건강관리)
- 진로교육 (인적자원개발 관점의 진로 설계)
- 부모교육 (신앙과 발달심리 통합)

### 3. ESG와 평생교육
- ESG 관련 평생교육 프로그램 (인적자원개발 관점)
- 지속가능한 조직문화와 웰니스
- **ESG평생교육전문강사 양성과정** (자격명: ESG평생교육전문강사, 제2024-006386호)

### 4. 보건 교육
- 생애주기별 건강관리 (간호학 전문성 기반)
- 웰니스 프로그램 (몸과 마음의 통합적 건강)
- 예방중심 건강교육
- 중독예방
- 자기돌봄

## 🛠️ 기술 스택

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Design**: 반응형 디자인 (Mobile-first)
- **Color Scheme**: 라벤더/보라색 계열 (따뜻하고 전문적인 분위기)

## 🚀 실행 방법

### 개발 환경 실행

```bash
cd wellness-coaching-institute
npm install
npm run dev
```

서버가 `http://localhost:5173`에서 실행됩니다.

### 프로덕션 빌드

```bash
npm run build
npm run preview
```
