# ✅ GitHub 저장소 생성 단계별 가이드

## 🎯 STEP 1: 새 저장소 만들기

### 1. GitHub 새 저장소 페이지 열기

다음 링크를 클릭하세요:
**👉 https://github.com/new**

또는:
1. GitHub 우측 상단 "+" 버튼 클릭
2. "New repository" 선택

---

## 📝 STEP 2: 저장소 정보 입력

다음과 같이 **정확하게** 입력하세요:

```
┌─────────────────────────────────────────────────────┐
│ Repository name *                                    │
│ wellness-coaching-institute                          │
│ ↑ 정확히 이 이름으로 입력하세요!                        │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ Description (optional)                               │
│ 웰니스코칭연구소 공식 홈페이지                           │
└─────────────────────────────────────────────────────┘

● Public          ← 이것 선택! (무료)
○ Private

❌ Add a README file        ← 체크 해제!
❌ Add .gitignore          ← 체크 해제!
❌ Choose a license: None  ← None 선택!
```

---

## ⚠️ 중요 주의사항

### ✅ 반드시 해야 할 것:
- Repository name: `wellness-coaching-institute` (정확히!)
- Public 선택
- **모든 체크박스 해제** (README, .gitignore, license)

### ❌ 하지 말아야 할 것:
- Private 선택 (유료일 수 있음)
- Add a README file 체크 (코드 충돌 발생)
- Add .gitignore 체크 (이미 있음)
- Choose a license 선택 (나중에 추가 가능)

---

## 🚀 STEP 3: 저장소 생성

**"Create repository"** 초록색 버튼 클릭!

---

## 📋 STEP 4: 생성 후 화면

저장소가 생성되면 다음과 같은 화면이 나타납니다:

```
Quick setup — if you've done this kind of thing before

https://github.com/binieni0611-maker/wellness-coaching-institute.git

…or create a new repository on the command line
…or push an existing repository from the command line  ← 이것!
```

---

## ✅ 저장소가 생성되었나요?

생성이 완료되면 저에게 알려주세요!

그럼 다음 명령어로 코드를 업로드하겠습니다:

```bash
cd /home/user/webapp
git push -u origin main
```

---

## 🆘 만약 오류가 발생하면?

다음 정보를 확인해주세요:

1. 저장소 이름이 정확한가? → `wellness-coaching-institute`
2. Public으로 생성했는가? → Yes
3. README를 추가하지 않았는가? → No (체크 해제)
